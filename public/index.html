Mobile Ticket Application

Paper Ticket Replacement

This mobile application was intended to replace the traditional paper ticketing system in place for J2 Acid Solutions LLC. The original system involved a lot of redundant back and forth between the field hand, office, and buyer. This made the paper tickets highly prone to costly errors. This ticketing system minimized user error by syncing with an online database and limiting what information could be changed.

Tools Used

  • Ionic 5
  • Angular 9
  • Firebase
